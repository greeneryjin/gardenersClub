# gardenersClub
식물 관리 기능을 제공하고 식물 집사님들과 식물팁을 공유하며 소통할 수 있는 식물 관리 & 커뮤니티 플랫폼입니다. 

사이트 기능
1. 커뮤니티 
- 팔로잉
- 사진
- 식물사전
- 식물 사전 리뷰 


2. 마이페이지
- 회원 수정
- 좋아요
- 스크랩


3. 로그인
- 네이버 
- 카카오
