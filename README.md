# gardenersClub
자신의 식물을 소개하고 소통하는 식물  커뮤니티 사이트 "gardenersClub"입니다. 

사이트 기능 
1. 커뮤니티 
- 팔로잉
- 사진
- 매거진
- 식물사전
- Q&A


2. 마이페이지
- 회원 수정
- 회원 탈퇴
- 좋아요
- 스크랩


3. 로그인
- 네이버 
- 카카오
