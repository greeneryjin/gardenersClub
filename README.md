# gardenersClub
식물 관리 기능을 제공하고 식물 집사님들과 식물팁을 공유하며 소통할 수 있는 식물 관리 & 커뮤니티 플랫폼입니다. 


사용 언어
```
- JAVA 11
- react 
```


사용 기술
```
- springboot
- Mysql
- S3
- Querydsl
- jpa
- jwt
- docker
- nginx
- AWS
- git action
```
